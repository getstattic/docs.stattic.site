Understanding the Structure

Let's understand what each directory does:

ğŸ“ content/ - Your Website Content

  • posts/: Blog posts written in Markdown
  • pages/: Static pages (About, Contact, etc.)
  • authors.yml: Information about blog post authors

ğŸ“ templates/ - How Your Site Looks

ğŸ“ assets/ - Static Files

templates: templates
assets: assets

# Styling
fonts:
  - Inter      # Body font
  - Poppins    # Heading font

# SEO settings
robots: public  # public or private

# Development settings
minify: false   # Set to true for production
watch: false    # Not implemented yet

Step 2: Understand Configuration Options

Setting Description Example
site_url Your website's URL https://example.com
site_title Site name in browser title My Blog
site_tagline Brief description Thoughts and tutorials
posts_per_page Posts on homepage 5
sort_by Post sorting method date, title, order
fonts Google Fonts to load [Inter, Poppins]
minify Compress CSS/JS true for production

Common Beginner Mistakes

1. Incorrect Front Matter Formatting

âŒ Wrong:

âœ… Correct:

date: "2025-06-19"

2. File Path Issues

âŒ Wrong:

âœ… Correct:

3. Markdown Formatting

âŒ Wrong:

**Bold text**with no spaces

âœ… Correct:

**Bold text** with proper spaces

4. Configuration Syntax

âŒ Wrong:

  Inter  # Missing dash
  Poppins

  - Inter

5. Missing Required Fields
