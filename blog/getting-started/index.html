Getting Started

May 21, 2025 · Robert

Welcome to Stattic -- a fast, Python-based static site generator for people who care about performance, control, and simplicity.

This guide will walk you through installation, project structure, and your first build.

Requirements

Python 3.7+

Make sure you have Python 3.7 or higher installed.

python3 --version

Install Dependencies

Clone the Stattic repo and install the required packages:

git clone https://github.com/getstattic/stattic.git
cd stattic
pip install -r requirements.txt

Dependencies Overview

Package Purpose
markdown Converts .md files to HTML
pyyaml Parses YAML front matter
Jinja2 Template engine
mistune Fast markdown parser
Pillow Image conversion to WebP
requests Downloads images and fonts
csscompressor Minifies CSS
rjsmin Minifies JS
watchdog (Optional) Enables future watch mode
tqdm Progress bar in terminal

Optional System Dependency

For converting animated GIFs to animated WebP:

# macOS
brew install webp

# Ubuntu/Debian
sudo apt-get install webp

Your First Page

Create a Markdown file in content/pages/about.md:

---
title: "About"
slug: "about"
template: page
---

Welcome to my site! This was built with Stattic.

Now you're ready to build.

Run Your First Build

From your project root:

python3 stattic.py --output output

This command:

  • Converts Markdown to HTML
  • Applies your templates
  • Outputs to the /output/ directory

Open output/index.html in your browser to view your site.

Common CLI Options

python3 stattic.py
  --output public
  --content content
  --templates templates
  --assets assets
  --site-title "My Stattic Site"
  --site-url "https://example.com"
  --fonts "Inter"
  --minify
Flag Description
--output Folder to build into (default: output)
--content Where your Markdown files live (default: content/)
--templates Folder for Jinja2 templates
--assets Folder for CSS, JS, fonts, etc.
--site-title Used in templates and metadata
--site-url Enables RSS and sitemap generation
--fonts Downloads Google Fonts and generates CSS
--minify Minifies CSS and JS assets
--posts-per-page Controls blog pagination (default: 5)
--sort-by Sorts posts by date, title, author, or order

Full CLI reference: See CLI Docs

Preview the Site Locally

Serve it with Python:

cd output
python3 -m http.server 8000

Visit http://localhost:8000 to see your site live.
